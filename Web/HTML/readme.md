# HTML

### 1. 简述

- HTML: Hyper Text Markup Language，超文本编辑语言，一种用于创建网页的标准标记语言。

- HTML 不是编程语言，是标记语言(markup language)

- 标记语言是一套标记标签(markup tag)

- HTML 运行在浏览器上，由浏览器来解析。
